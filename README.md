# Middletown Township Floodmapper

Middletown Township flood mapper configured with the official NJ municipal boundary, Sea Bright gauge data, and Bunny-hosted raster overlay roots.

Depth overlays are expected at:

`https://floodmapperv1.b-cdn.net/DepthPNGs/MiddletownTownship/`

Stage overlays are expected at:

`https://floodmapperv1.b-cdn.net/StagePNGs/MiddletownTownship/`

Historical top-tide data include Hurricane Sandy using NOAA CO-OPS Sandy Hook
station 8531680: 13.231 ft MLLW at 2012-10-29 19:36 local, converted with
NOAA station datums to 10.411 ft NAVD88.
